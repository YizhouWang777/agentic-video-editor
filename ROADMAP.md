# Roadmap

Agentic Video Editor is alpha software. The roadmap is organized around agent
reliability rather than UI features.

## Near Term

- Richer EDL: tracks, overlays, music beds, crop, speed, and transitions.
- Subtitle OCR QA and safe-zone checks.
- Batch mode for many videos using the same template.
- Better transcript import/export formats.

## Runtime Adapters

- HyperFrames adapter for HTML/CSS/GSAP motion scenes.
- Remotion adapter for React templates.
- Adapter contract for generated clips from external APIs.

## Asset Understanding

- Better keyframe selection.
- Scene boundary detection.
- Visual notes from multimodal models.
- Music beat/BPM analysis.

## Agent UX

- More actionable failure explanations.
- EDL diff and undo.
- Reusable template packs.
- Skill packaging for Codex and Claude Code.
